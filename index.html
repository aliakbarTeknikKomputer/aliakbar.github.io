Proyek & Praktikum

Dokumentasi pengalaman praktis pengerjaan proyek IT (Semester 1-2)

IoT Alat Deteksi Telur Busuk

  • Merancang arsitektur hardware berbasis Arduino Uno dan mengintegrasikan sensor LDR.
  • Melakukan kalibrasi sensor untuk membaca tingkat intensitas cahaya pada telur secara otomatis.
  • Memasang layar LCD I2C untuk menampilkan status kualitas kebusukan telur secara real-time.

Implementasi ZeroTier VPN

  • Konfigurasi Virtual Private Network (VPN) Mesh menggunakan platform ZeroTier One.
  • Menghubungkan konektivitas data dan komunikasi jarak jauh terenkripsi antar perangkat.
  • Melakukan pengujian routing aman untuk memvalidasi kestabilan lalu lintas data jaringan.

Clickjacking Checker Tool

  • Mengembangkan script tools penguji celah keamanan web berbasis Python Flask Framework.
  • Mengimplementasikan kontainerisasi sistem menggunakan Docker untuk efisiensi deployment.
  • Menganalisis respon HTTP Headers keamanan seperti X-Frame-Options pada target domain.

Jaringan Nirkabel (Wireless)

  • Melakukan instalasi perangkat keras serta manajemen konfigurasi Access Point lokal.
  • Menerapkan standar protokol enkripsi nirkabel tingkat lanjut (WPA2 dan WPA3).
  • Mengoptimalkan distribusi pembagian frekuensi sinyal wifi untuk meminimalkan interferensi.

Simulasi Jaringan Cisco

  • Mendesain topologi arsitektur jaringan komputer enterprise dengan Cisco Packet Tracer.
  • Konfigurasi routing statis, switching, distribusi alokasi VLAN, serta inter-VLAN routing.
  • Mengatur manajemen pembagian subnetting IP Address (VLSM) untuk efisiensi node jaringan.

Pembuatan Media Canva

  • Memanfaatkan Canva untuk mendesain diagram visual alur sistem jaringan komputer secara detail.
  • Menyusun dokumentasi tata letak laporan praktikum teknis agar terstruktur dengan rapi.
  • Merancang media presentasi infografis interaktif untuk pemaparan materi akademik kelompok.
